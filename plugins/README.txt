This directory contains RoadRunner plugins and example plugins (in examples folder)

List of RoadRunner Plugins:
1. rrp_lm.dll
This supports the Levenberg-Marquardt optimizer for fitting data to models.
See libroadrunner.org for details.

2. rrp_add_noise.dll
This plugin can be used to generate Gaussian noise and add it to RoadRunner data. This is useful for generating synthetic data sets to test
various algorithms, e.g. parameter fitting algorithms.
