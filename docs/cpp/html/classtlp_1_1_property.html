Detailed Description

template<class T>
class tlp::Property< T >

Template class implementing a PluginProperty.

The characteristics of a Plugin Property is its type and its value. Various functions assist in setting/getting the properties value. Its base class, PropertyBase, encapsulate the properties name, hint and description.

Of importance is the ability to get a pointer to its internal value, e.g. getValuePointer(), or getValueHandle(), as it can be used as a handle in the derived Plugins C API.
